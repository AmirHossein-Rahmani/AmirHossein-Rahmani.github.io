# AmirHossein Rahmani - Economist Portfolio

## Overview
Professional portfolio website for AmirHossein Rahmani, featuring an animated tree-code background effect and clean, modern design suitable for an economist professional.

## Features
- Animated tree-code background using HTML5 Canvas
- Rain animation with falling symbols (%, $, #, @, &, 1, !)
- Responsive design with Tailwind CSS
- Smooth scrolling navigation
- Professional sections: Hero, About, Expertise, Contact
- Fixed header with scroll effect
- Professional slide-in animations

## Structure
- `index.html` - Main HTML with metadata and imports
- `app.js` - Main React application component
- `components/` - Individual React components for each section
- `utils/` - Tree and rain animation logic
- `styles/` - CSS animations

## Design
- Primary Color: Black (#1a1a1a)
- Secondary Color: Light Gray (#f5f5f5)
- Accent Color: Dark Gray (#404040)
- Monochrome color scheme inspired by professional portrait
- Classic French typography with EB Garamond font

## Animation
- Background features a tree-code animation with floating characters
- Rain effect with shiny white symbols falling
- Professional slide-in animations for hero content
- Canvas-based name drawing effect
